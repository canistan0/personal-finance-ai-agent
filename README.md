# Personal Finance AI Agent

## Overview

This project is a Python-based AI agent that analyzes personal financial data stored in a CSV file. It allows users to ask natural language questions about their spending and receive structured, meaningful answers.

The system uses a modular tool-based architecture including:

- CSV data reader
- calculator functions
- AI agent logic
- optional web context module

---

## Features

- Analyze spending by category (food, transport, etc.)
- Perform percentage and sum calculations
- Natural language CLI interface
- Tool-based agent architecture
- Testable modular components

---

## 🧠 Architecture

User Input → AI Agent → Tools (CSV Reader / Calculator / Web Context) → Structured Answer
